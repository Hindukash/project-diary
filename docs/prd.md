# Product Requirements Document (PRD)

## User Stories

### Content Management
* As a user, I can create, edit, and delete new project entries with a title and a rich text body.
* As a user, I can organize my project entries by creating sub-topics or nested sections.
* As a user, I can view the history of changes made to an entry so I can track my progress over time.

### User Interface
* As a user, I can upload images to my notes.
* As a user, I can switch the UI between a dark and light theme.
* As a user, I want a simple, modern, and user-friendly interface with modern drag-and-drop elements.

### Search & Organization
* As a user, I can add tags to my entries to categorize them.
* As a user, I can perform advanced searches on my entries, filtering by tags, dates, or keywords.

### Data Handling
* As a user, I can export an entry or an entire project as a file (e.g., PDF or Markdown).
