# Project Diary: Project Brief

## Project Name
Project Diary

## Goals
To create a diary-style, web-based notepad where I can keep track of my studies, tutorials, and workflows. The application should allow me to make notes, upload pictures, and save these entries for future use. The UI should be a simple, user-friendly, and modern, with a dark design and drag-and-drop elements.

## Target Users
A developer who wants to track projects, notes, and progress in a personal, structured way.
